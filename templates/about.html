🧠 About Kansi AI

Technology should be kind. We use machine learning to gently identify emotional patterns in text — not to label anyone, but to open a door to awareness and support.

🌱 How It Works

You write freely. Kansi AI reads the emotional texture of your words using TF-IDF feature extraction and a tuned Logistic Regression model trained via supervised learning. It looks for linguistic patterns research has linked to distress.

🔒 Security

Fernet-encrypted storage for all sensitive data. Salted password hashing. Rate limiting on all endpoints. Input sanitisation with bleach. Row-level security enforced at the query level. No API keys in frontend code.

🤝 Our Promise

Kansi AI will never pretend to be a therapist. It is a screening tool — a first step. For real support, speak with a professional. If crisis language is detected, our system silently connects with emergency services to keep you safe.
